Quick Start

1. Install BioCode

# Clone the repository
$ git clone https://github.com/yourusername/biocode.git
$ cd biocode
# Install dependencies
$ pip install -r requirements.txt

2. Launch Your First Agent

# Start monitoring a project
$ python -m biocode.agent --project /path/to/your/project
# Or use the Python API
from biocode import BioCodeAgent
agent = BioCodeAgent("/path/to/project")
agent.start()

3. Monitor with Dashboard

# Start the web dashboard
$ python launch_dashboard.py
# Open in browser
